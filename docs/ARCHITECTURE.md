## 关键技术决策

| 决策 | 理由 |
|------|------|
| 串行处理 (非并发) | Pollinations 限制每 IP 1 并发，并发反而触发 429 |
| curl 而非 urllib | curl 子进程更稳定，urllib 频繁连接失败 |
| 每个 step 独立跳过 | 重跑时不必从头开始，省时间 |
| ffmpeg zoompan | Ken Burns 推镜效果让静态图有动感 |
| 中文字幕烧录而非外挂 | 兼容所有播放器，包括 B 站网页 |
